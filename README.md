# 🎓 Student Management System API

## 📌 Description
This is a RESTful API built using ASP.NET Core for managing student data.  
It supports CRUD operations and follows clean architecture principles.

---

## 🚀 Features
- Add Student
- Update Student
- Delete Student
- Get All Students
- Get Student by ID
- JWT Authentication (if added)

---

## 🛠️ Tech Stack
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- C#

---

## 📂 Project Structure
- Controllers
- Services
- Repositories
- DTOs
- Models

---

## ⚙️ How to Run
1. Clone the repository  
2. Setup SQL Server Database  
3. Update Connection String in appsettings.json  
4. Run the project
